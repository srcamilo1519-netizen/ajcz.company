## [1.0.0] - 2024

### Added
- ✨ Landing page completa con animaciones CSS
- 🎭 Preloader animado con partículas flotantes
- 🌐 WebView integrado con PyQt6 WebEngine
- 📱 Botón de WhatsApp que abre app nativa
- 🎨 Diseño cyberpunk con paleta oscura (#050d1a, #64ffda)
- ⚡ Scroll reveal animations
- 🍔 Menú hamburguesa responsive
- 📄 README con badges y documentación completa
- 📜 LICENSE MIT
- 🔧 Script de build con PyInstaller

### Technical
- Python 3.9+ con PyQt6 6.7+
- Animaciones CSS puras sin librerías externas
- Fuentes: Syne, DM Sans (Google Fonts)
- Estructura modular: HTML + Python separados
